    # FLEX CONTAINER 속성
  
    - flex-direction(row): item의 배열 방향
    row|row-reverse|column|column-reverse
  
    - flex-wrap(nowrap): item을 감쌀 것인지 여부 
    nowrap|wrap|wrap-reverse
  
    - flex-flow(row nowrap): flex-direction flex-wrap 속기
  
    - justify-content(flex-start): item의 수평정렬(flex-direction: row), 수직정렬(flex-direction: column)
    flex-start|flex-end|center|space-between|space-around|space-evenly
  
    - align-items(stretch): item의 수직정렬(flex-direction: row), 수평정렬(flex-direction: column)
    stretch|center|flex-start|flex-end|baseline
  
  
    # FLEX ITEM 속성
    
    - flex-grow(0): item의 크기 증가
    number
    
    - flex-shrink(1): item의 크기 측소
    number
    
    - flex-basis(auto): item의 넓이
    number|auto
    
    - flex(0 1 auto): flex-grow flow-shrink flex-basis 속기
  
    - order(0): item의 정렬 순서
    number
