Document Structure

    1. html
    root element
    

    2. head
    meta data (data for document)

    1) title
    document title
    
    2) description
    document description

    3) charset 
    document character set. UTF-8 is default.
    
    * UTF-8 (Unicode Transformation Format - 8 bit)
    unicode is world standard for text and emoji.
    It covers all characters in the world.

    4) external source link (css, js files)

    5) other meta data


    3. body 
    document contents
