display

    엘리먼트의 진열 방법

    1 block
    2 inline-block
    3 inline
    4 none

display: block

    부모 엘리먼트의 넓이를 모두 차지한다
    div, p, nav등 대부분 엘리먼트의 기본값
item
item

display: inline-block

    컨텐츠에 필요한 만큼만 넓이를 갖는다
    buttons, input등의 기본값
item
item

display: inline

    컨텐츠에 필요한 만큼만 넓이를 갖는다
    span, a등의 기본값

    엘리먼트가 width, height 속성을 가질 수 없다
    y축 padding, margin 속성이 적용되지 않는다
    레이아웃 용으로 적합하지 않다
item
item

display: none

화면에 나타나지 않는다
