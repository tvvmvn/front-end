window object

toolkit for web development.

message or confirm

role as user interface.

  • window.alert()
  • window.confirm()

get user information

to develop app based on user information for example, location based app and device optimized app.

  • window.navigator.geolocation
  • window.navigator.userAgent

browsing history

back button in application.

  • window.history

request data

AJAX request

  • window.fetch()
  • window.XMLHttpRequest()

web storage

save data in web browser. for instance, login token, theme, shopping cart and etc.

  • window.document.cookie
  • window.localStorage

view development

  • window.document
  • window.document.documentElement
  • window.innerHeight
